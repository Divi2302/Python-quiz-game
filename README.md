Python Quiz Game:
This is a simple Python-based Quiz Game. The game asks the player a series of questions, evaluates the answers, and provides a score at the end.

Features:
Multiple-choice questions.
Score tracking system.
Easy-to-extend for adding more questions.
easy quit.

Requirements:
Python 3.x

How to Play:
Upon running the game, you will be asked a series of questions.
Each question will have multiple choices.
Type the letter corresponding to the correct answer and press Enter.
At the end of the quiz, your score will be displayed.
